Network Speed Test (NDT)

Our app includes a Speed Test feature powered by the Network Diagnostic Tool (NDT) protocol provided by Measurement Lab (M-Lab).

When you initiate a speed test, certain data is collected and transmitted to M-Lab's servers, including:

  • Your IP Address
  • Network performance metrics (Download/Upload speeds, latency, etc.)

Important: M-Lab is an open-source research project. In order to support global internet research, M-Lab publishes all test results, including your IP address, in their public database for open access. By using the speed test feature, you consent to the collection and public release of this data.

For more information on how M-Lab handles your data, please review the M-Lab Privacy Policy.
