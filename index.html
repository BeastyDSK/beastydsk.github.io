1. Network Diagnostic Tests (NDT7) & M-Lab

⚠️ Critical Notice Regarding Speed Test Data This Application runs network performance tests (Speed Tests) using the NDT7 protocol maintained by Measurement Lab (M-Lab).

When you run a speed test in this Application, the test is conducted between your device and an M-Lab server. M-Lab collects specific data to promote scientific research on Internet performance.

What is collected by M-Lab?

  • IP Address: The public-facing Internet Protocol (IP) address of your device is collected to understand network topology.
  • Performance Data: Speed, latency, jitter, and other network diagnostic metrics.

Is this data private?

No. To support open Internet research, M-Lab publishes all test data publicly. By running a test, you acknowledge that your IP address and test results will be included in M-Lab's public dataset. M-Lab does not collect personal information like your name, email, or street address.

For more details, please review the M-Lab Privacy Policy.
